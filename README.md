# Vehicle Service Booking System

A full-stack web application that enables users to book car or bike services and allows mechanics to manage incoming service requests.

---

## Project Architecture

This project is built using the Clean Architecture principles on the backend and a modern React + TypeScript frontend.

---

## Features

For Users:
- Register and log in securely
- Book car or bike service appointments
- Choose service type and input vehicle details

 For Mechanics:
- View incoming service requests
- Accept or reject requests
- Update service status: In Progress / Completed

---

## Tech Stack

 Backend (.NET Clean Architecture):
- ASP.NET Core Web API
- Entity Framework Core
- Clean Architecture structure (Domain, Application, Infrastructure, Presentation)
- SQL Server

Frontend (React + TypeScript):
- React with TypeScript
- Axios for API communication
- React Router for routing
- React Hook Form / Yup for form validation (if used)
